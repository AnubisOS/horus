## Installation

Download the repo using
```git
  git clone https://github.com/HushmKun/horus
```
Then 
```bash
  sudo make install
```
    

## Usage/Examples

```Bash
horus
```


## Features

- Pure python (No dependecies).
- Relatively fast.
